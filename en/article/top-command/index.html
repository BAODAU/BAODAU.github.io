Linux memory types#

There are three types of memory, and one is optional.

  1. physical memory, a limited resource where code and data must reside when executed or referenced.
  2. optional swap file: where modified (dirty) memory can be saved and later retrieved if too many demands are made on physical memory
  3. big boy virtual memory: unlimited resource intended to
  • abstraction: free from physical memory addresses/limits
  • isolation: every process in a separate address space
  • sharing: single mapping can serve multiple needs
  • flexibility: assign a virtual address to a file
