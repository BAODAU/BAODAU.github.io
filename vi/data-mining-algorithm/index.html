Data Mining Algorithm

February 18, 2020 · 1 min · baoqdau

The score functionused to measure the quality of different tree structures is a general miscalassification loss function, defined as $\sum_{i=1}^nC(y(i),\hat{y}(i))$

  • $C(y(i),\hat{y}(i))$ is the loss incurred when the class label for the ith data vector, y(i) is predicted by the tree to be $\hat{y}(i)\ne y(i)$

cross-validation#

  • partitions the training data into a subset for building the tree then estimates the misclassifcation rate on the remaining validation subset.
  • this partitioning is repeated multiple times on different subsets and the misclassfication rates are then averaged to yield a cross-validation estimate of how well a tree of a particular size will perform on new, unseen data.

characteristics#

  • estimate the performance of any tree model on data not used in the construction of the tree $\rightarrow$ providees an estimate of generalization performance.
